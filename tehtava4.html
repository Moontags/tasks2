Array Operations in JavaScript

A. JavaScript Reference for Array Handling

A good JavaScript reference for array handling is the MDN Web Docs - Array.

B. Mutating an Array In Place

When an operation mutates an array in place, it means that the operation directly modifies the original array. No new array is created; the existing array object is changed.

Examples of Mutating Operations:

  • push()
  • pop()
  • shift()
  • splice()
  • sort()

C. Returning a Copy of an Array

When an operation returns a copy of an array, it means that the operation creates a new array object containing the elements (potentially modified) of the original array. The original array remains unchanged. The result of the operation is a reference to this new array.

Examples of Operations Returning a Copy:

  • slice()
  • concat()
  • map()
  • filter()
  • reduce() (often returns a single value, but can return a new array)
  • Spread syntax (...) in array creation (e.g., const newArray = [...oldArray, newElement];)
